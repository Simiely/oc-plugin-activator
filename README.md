# OC插件 快速激活工具

[![Build Windows EXE](https://github.com/Simiely/oc-plugin-activator/actions/workflows/build-exe.yml/badge.svg)](https://github.com/Simiely/oc-plugin-activator/actions/workflows/build-exe.yml)

一键清空 OctaneRender 残留文件 + 复制资源文件的 Windows 工具。

**使用：** 把 exe 和 `thirdparty`、`OctaneRender`、`octane` 放一起 → 填用户名和 C4D plugins 目录 → 先②清空再③复制。

**下载：** [Actions 页面](https://github.com/Simiely/oc-plugin-activator/actions) → 最新成功构建 → Artifacts
